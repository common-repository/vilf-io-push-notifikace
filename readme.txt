Vilf.io je aplikace a plugin pro rozesílání push zprávy na desktop i smartphony – přímo do zařízení uživatele. Umožní vám upozornit návštěvníky vašeho webu na opuštěný košík, akční nabídku, nový článek nebo doručenou zásilku.

== Description ==

= Vilf.io – vaše tajná zbraň o pozornost zákazníků =

* snadné propojení s **WooCommerce**
* kompatibilní s každou **WP šablonou**


Webové push notifikace jsou zprávy, které se (např. na rozdíl od newsletterů) **posílají odběrateli přímo do jeho zařízení**, nezávisle na tom, zda je právě na vašem webu nebo ve své e-mailové schránce.

Push zprávy vám pomohou **získat pozornost a zapojení návštěvníků** vašeho webu a přimějí je k odeslání objednávky či poptávky. Vilf.io využijí **e-shopy**, obsahové weby i B2B společnosti.

Se stejným úsilím, jaké věnujete tradičním marketingovým kanálům, vám notifikace přinesou **několikrát vyšší návštěvnost** a zlepší váš zákaznický servis.

= Co vám plugin Vilf.io umožní? =
* **personalizaci** – ve zprávě můžete registrovaného uživatele oslovit jménem,
* poslat uživateli **upozornění na opuštěný košík** (využíváte-li WooCommerce),
* **odeslat push zprávu o změně stavu objednávky** (např. předání balíčku dopravci),
* **zdarma rozesílat neomezený počet odeslaných notifikací** až **1 000 odběratelům**.

== Installation ==
= 1) Integrujte Vilf.io na svůj web =

První krokem je registrace do aplikace na [https://vilf.io/sign/up](https://vilf.io/sign/up) a aktivací pluginu ve vašem redakčním systému

1. Nainstalujte plugin ze stránky WordPress.org nebo jej nahrajte do vaší složky wp-content/plugins.
2. Aktivujte Vilf.io plugin.
3. Vložte do nastavení Vilf.io pluginu ID webu a API token, jenž najdete na v administraci Vilf.io.
4. Povolte funkce, které chcete používat v nastavení Vilf.io pluginu.

= 2) Získejte odběratel =

V administraci vytvořte 1krokový nebo **2krokový dotaz** pro získání souhlasu návštěvníků svého webu se zasíláním novinek. Můžete si vybrat, kdy jim přihlašovací banner zobrazíte: po 5 minutách návštěvy, po vstupu na určitou stránku nebo třeba při odchodu z webu.

= 3) Segmentujte a tvořte cílené zprávy =

**Získané kontakty** můžete rozdělit podle typu používaného zařízení, operačního systému, poslední návštěvy nebo podle tagů. Vytvořené segmenty **využijete k doručení personalizovaných zpráv** uživatelům.

= 4) Doručte notifikace =

Plugin vám pomůže vytvořit **automatizované upozornění na opuštěný košík** a doručenou zásilku. Je také možné rozesílat zprávy s odkazem na nové články nebo videa.

= 5) Vyhodnotíte účinnost kampaní =

V přehledu výsledků rychle zjistíte, jak si vaše kampaně vedou: Vilf.io vám nabídne informace o míře zobrazení a prokliků z notifikací nebo o efektivitě žádostí o odběr.
